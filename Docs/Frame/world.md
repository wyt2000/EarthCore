# 游戏世界观

## 背景故事

```text
古老的种族在冰河时期的到来之时运用炼金术和科技的结合打通了通往地下的大门,
利用岩浆的光和热在地底世界构建起了伟大的文明.

然而一次地壳运动(某种浩劫)导致通往地面的通道彻底关闭,文明支离破碎,科技遗落,
分散在各地的族人发现地底每隔一段距离就存在空腔(初步想法是每一次冰期到来都有一个文明会逃入地下,空腔来自上一个文明),
他们的技术随无法一口气回到地底,但却能尽力打通这些空腔,种族想利用不断向上寻找并打通空腔回到地表.

空腔中遗落着各类魔法、科技、神秘力量,抵达不同空腔的族人分支出了不同的族群.

就在最接近地表的空腔被打通的时候,该空腔中的诅咒突然被释放(还没想好到底是什么来源,以后再说),
种族中的一个分支族群利用某种技术牺牲了自己,封印了通往地表的最后空腔,地底世界被关闭.

多年后冰河时代再次降临,新的文明(可以是人类)通过遗落的资料得知了部分关于空腔的信息,开发出探测空腔密度的手段,
企图在空腔竖向排列最密集的区域打通通往地底的通道(否则时间与技术不允许他们抵达宜居层),
然而钻地机器却在进入地底后不久就神秘失踪了,调查队来到失踪地点的洞口准备探索,
主角就是其中一员(主角的初步设定是当年封印深渊的族群的后裔).
```

## 设定集

### 初代文明

```text
在不断向上探索的过程中的同时分化出了不同的阶级,越往上元素越富集,资源越丰富,
靠近上方的族群就掌握更多高级资源,靠近底层的族群只能靠为上层开采岩浆等基础资源、贡献廉价劳动力维生.

等到最后阶段,上层探索阶级基本被异化,他们不想到达地表后和所有人平等地共享资源,
因此动用某种手段(诅咒来源)企图把整个种族困在地底,少部分人去往地表.
```
