# 游戏世界观

## 背景故事

```text
故事发生在一片古老的架空大陆，这里似乎有过数次辉煌的文明，但似乎都因某种原因遗落了，只留下极其有限的科技和史料记载。
从这些记载中，人们得知了前几代文明的毁灭原因——周期性到来的冰河时代。同时，人们也逐渐掌握了记载中描述的元素核心制造技术，利用元素核心发展出了强大的元素机械。
建立起了伟大的文明。

不久，冰河时代如期而至，诸多元素核心开始熄灭，文明到达了生死存亡之际。就在此时，以为科学家声称检测到地底存在一系列的空腔，能够直达地下宜居带，
人们便集中力量制造了大型钻地探测器洛斯特艾斯号，企图打通各个空腔，建立地下文明。

然而，洛斯特艾斯号钻入地底后不久便凭空消失，随行的总工程师恩杰尼·切普罗尔以及诸多随行人员全部失去联络，政府多次组织救援队都无功而返，整个文明陷入动荡与恐慌。

21岁的伊克斯·切普罗尔是失踪工程师恩杰尼·切普罗尔的儿子，从小活泼好动，喜欢鼓捣元素机械。在得知父亲失踪，政府搜寻无果后，决心自行前往空腔开启地寻找父亲下落。在准备行李的途中，他偶遇了正在召集民间救援队的退伍老兵弗姆·欧提米加，随即与发小克莱姆·斯达伊夫加入了救援队，一同参加的还有一位自称是科学家的古怪老头。临行之前，伊克斯的青梅竹马莫西·诺戈尔曼挡住了他们的去路，愤怒地阻止二人去做如此危险之事。但伊克斯一改往日对莫西打马虎眼的态度，决意要去。莫西一气之下决定和四人一同前往。

于是，一行人来到了空腔的开启地，开始了一段惊心动魄的冒险......
```

## 设定集

### 元素之力与驭素师

```text
整个种族的科技是基于元素之力驱动的，只有具备对应品质的人才能掌握相应的元素，掌握相应的元素才能为对应的元素机械、装备等物品注入元素之力，这种人被称为【驭素师】。

被注入元素之力的物品可以消耗元素之力来运作（可以理解为元素是一种资源，元素机械是消耗这种资源运作的，而只有驭素师才能收集元素并为元素机械注入元素）
五种元素分别对应五种精神力量：金——坚定；木——善良；水——智慧；火——热情；土——冷静。

但各个文明的人们并不知道如何掌握元素之力，他们认为驭素师的能力是和血统绑定在一起的，因此驭素师往往有着庞大的家族体系和极高的社会地位。

每个驭素师往往都只能掌握一到两个元素，极少有能掌握三到四个元素的驭素师出现，从未出现过能同时掌握五个元素的驭素师。
```
### 初代文明

```text
由于驭素师家族对整个文明的重要性，驭素师逐渐成为了文明的统治阶级，他们领导整个文明不断向上探索，开拓着地底世界的边界。
在不断向上探索的过程中，人们发现越往上元素越富集，住在上层的人变能依赖富足的元素过更好的生活，但他们又不得不依赖底层的人操纵元素机械为他们提供各类生活必需品。
文明内部渐渐分化出了等级制度，驭素师家族高高在上，其他人卑如蝼蚁。

继续向上探索，人们渐渐发现地底世界的最上方通往一片平坦广阔的新世界，那里的元素丰富无比，且均匀分布在地表，是一片自由的天堂
于此同时，驭素师家族中出现了越来越多的无法掌控元素之力的人，而原本不具备驭素能力的底层人中却不断涌现出新的驭素师。
由于害怕到达地表后自己享受的特权消失，同时也害怕自己的地位被取代，各个驭素师家族决定不打通通往地表的道路，同时异化元素之力，用于感染普通人
他们声称可以用后天技术将普通人培养为驭素师，表面让普通人具备驾驭元素之力的能力，实际却令他们感染被异化的元素之力，令普通人完全处于他们的控制之下。
（意思就是驭素师通过把元素变为一种类似药物的东西注入给普通人，令普通人短时间获得驾驭元素的能力，但想要一直获得这种能力，需要驭素师不断给他们提供药物）
这种被异化的元素之力被称为【元素诅咒】：金——多疑；木——利己；水——物欲；火——暴力；土——色欲；

然而，令他们没想到的是，元素诅咒渐渐失去了控制，开始侵蚀人的理智，让人变成被欲望驱使的奴隶。（根本原因是元素诅咒会不断利用附近的元素之力生成自己）
同时，元素诅咒在统治阶级内部也开始蔓延，激发了内斗。部分人主张抛弃被污染的人去往地表，但由于统治阶级内部也有大量感染者，
诸多家族主张将被感染者隔离研究解决办法。双方大战一触即发。

此时，一支来自底层的驭素师队伍出现了，他们当中竟然有能够同时掌握五种元素的驭素师，他们似乎具备免疫元素诅咒的能力，但却始终没有找到解除元素诅咒的方法，
他们带领未被感染的人们开启了通往地表的通道，封印了地底世界，尘封了元素之力相关的记载，将元素机械作为技术黑盒保留了下来，开启了新的文明，直到冰河世纪的到来

```

### 切普罗尔家族

```
古老的元素机械工匠家族切普罗尔。家族世代以制作元素机械为生，掌握着重要的火元素与金元素驱动的机械设计图纸。
元素机械的能量来源是元素核心，古老的图纸虽然提供了元素核心的提炼方法，却从未解释元素核心从何而来，元素核心的来源被隐藏在了繁琐复杂的步骤当中。
切普罗尔的家族的祖先切·普罗尔便是最早能同时掌握五种元素的驭素师，他封印了地底世界。探索空穴的钻地机器洛斯特艾斯号便是由该家族打造。
```

### 洛斯特艾斯号

```
冰川危机到来后，切普罗尔家族主持制作的大型元素机械。用以钻探、串联空穴。
被寄以厚望的洛斯特艾斯号在首次启用，进入空穴后不久便完全失联，
同时失踪的还有总工程师恩杰尼·切普罗尔等随舰人员。
```

### 元素核心

```
元素核心是元素机械的关键部件。元素核心实质上是对元素诅咒和元素之力的整合平衡，
用克制元素诅咒的元素之力封装元素元素诅咒，使其既能驱动元素之力，又不会污染元素之力。
（例如用水元素之力封装火元素诅咒，形成的火元素核心就能驱动核心附近的火元素）
```

### 元素机械

```
分为古代元素机械和现代元素机械，古代元素机械大多为空穴内的遗物，需要靠直接注入元素之力驱动
现代元素机械则依赖元素核心驱动。
```

### 第六元素

```
第六元素，主角能够拥有净化元素诅咒的原因，本质上在于他拥有五种纯粹的精神力量，
同时将这五种精神力量注入到了第六种未知的精神力量中，这中精神力量不会明确告知玩家，
属于开放式表达，但游戏主要想表达的精神力量为好奇心、探索欲。
```



### 角色

```text
主角（火） 伊克斯·切普罗尔
21岁，来自古老的元素机械工匠家族。从小精力旺盛、喜欢冒险。性格开朗活泼。
年幼时母亲意外丧生，父亲恩杰尼·切普罗尔希望他能努力学习锻造技术，将来继承家业，他却沉迷利用元素核心打造各种稀奇的装备，用于辅助自己探险。
在空穴钻探任务中，他的父亲和钻地机器洛斯特艾斯神秘失踪，政府几次派出救援队都无功而返，他便参加了老兵弗姆组织的民间救援队。
似乎拥有操纵元素和净化污秽的力量

女队医（木）莫西·诺戈尔曼
22岁，来自古老的医师家族诺戈尔曼。主角的青梅竹马，性格善良，表面脾气暴躁，总是骂伊克斯和小胖四处闯祸，
但还是会帮他们包扎伤口，收拾烂摊子。处处管着伊克斯，像是介于他姐姐和母亲之间的角色。
在伊克斯和小胖加入民间救援队后劝阻无果，索性跟随一起加入。

老兵（金）弗姆·欧提米加
44岁，政府军成员，性格坚毅的硬汉形象，洛斯特艾斯号钻机失踪后的第一批救援队成员，
他声称第一批救援队并未像政府宣布的那样无功而返，而是几乎全军覆灭。
他在战友的拼死相救下才活了下来，他一再要求政府组织成员再次前往救援，却都杳无音讯，甚至被开除了军籍。
无奈之下他只好自己召集救援队

厨师小胖（土）克莱姆·斯达伊夫
21岁，主角的好基友。表面浪贱风骚，经常和莫西斗嘴，遇事沉着冷静，值得信任

老科学家（水）波顿·温斯伊尔
64岁，研究元素之力的老科学家，厌倦社交，沉默寡言，博学智慧的智者形象，
由于长期从事元素之力研究，被学术圈主流的炼金派（主张对元素核心不求甚解的应用派）排挤出了学术圈，
落魄潦倒，但始终坚定不移地从事研究。因怀疑洛斯特艾斯号的失踪和元素之力有关，加入了老兵弗姆组建的民间救援队。
```
